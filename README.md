# Wiper

This project has a D language source file (`main.d`).

## Getting Started

1. **Requirements:**
   - D compiler (e.g., [DMD](https://dlang.org/download.html), LDC)
   - Windows OS (needed)

2. **Build and Run:**
   - Open a terminal in this directory.
   - Compile:
     ```powershell
     dmd main.d
     ```
   - Run:
     ```powershell
     .\main.exe
     ```
     Run as admin


## Project Structure
- `main.d`: Main source file for the application.
